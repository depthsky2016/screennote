## 描述
「截图笔记」让你的每一张截图都变成有价值的笔记。

截图之后，导入到截图笔记，系统会自动识别图中文字，并用 AI 生成标题、摘要和标签。从此告别截图堆积如山、翻找困难的问题。

### 核心功能
- 截图即笔记：导入后自动生成结构化笔记
- OCR 文字提取：精准识别中英文内容
- AI 智能摘要：自动总结截图重点并推荐标签
- 全文搜索：搜索标题、摘要、原文与标签
- 标签分类：支持自动和手动整理
- 隐私优先：OCR 本地执行，仅 OCR 文本发送到 Kimi API
