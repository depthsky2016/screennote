截图笔记 隐私政策

最后更新：2026 年 4 月 7 日

核心原则：截图笔记将你的隐私放在首位。所有截图数据均存储在你的设备本地，我们不会收集、上传或共享你的个人数据。

1. 数据收集

截图笔记 不会 收集以下任何数据：

  • 个人身份信息（姓名、邮箱、电话等）
  • 设备标识符或追踪数据
  • 使用行为分析数据
  • 位置信息

2. 本地数据存储

你导入的截图、OCR 识别结果、AI 摘要和笔记内容均存储在你的 iPhone 本地，使用 Apple 的 SwiftData 框架管理。这些数据不会被上传到任何服务器。

3. 第三方服务

截图笔记使用以下第三方服务：

  • Kimi API（月之暗面）：当你配置了 API Key 后，截图中的 OCR 文字内容会发送至 Kimi API 以生成智能摘要。此过程仅传输文字内容，不传输图片本身。你可以随时在设置中删除 API Key 以停止此功能。
  • Apple Vision Framework：OCR 文字识别完全在设备本地执行，不涉及任何网络传输。

4. 相册访问

截图笔记需要访问你的相册以导入截图。我们仅读取你主动选择的图片，不会扫描或访问相册中的其他内容。

5. API Key 安全

你的 Kimi API Key 通过 Apple Keychain 安全存储在设备本地，采用硬件级别的加密保护。API Key 不会被发送到我们的服务器或任何第三方（除 Kimi API 本身外）。

6. 数据删除

你可以随时在 App 内删除任何笔记。卸载 App 将永久删除所有本地数据。

7. 儿童隐私

截图笔记不面向 13 岁以下儿童，也不会故意收集儿童的个人信息。

8. 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在此页面发布，并更新顶部的"最后更新"日期。
