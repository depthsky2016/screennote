常见问题 / FAQ

Q: 如何从相册导入截图？
How do I import screenshots from Photos?

打开首页，点击右上角的「+」按钮（Mac 上按 ⌘N），选择一张或多张截图导入。导入后应用会自动用 Apple Vision 进行本地 OCR 文字识别。

Tap the "+" button on the top right of the Home tab (⌘N on Mac) and select one or more screenshots. The app will run local OCR via Apple Vision after import.

Q: OCR 文字识别会上传图片吗？
Does OCR upload my images?

不会。OCR 完全在设备本地运行，使用 Apple Vision Framework。图片永远不会离开你的设备。

No. OCR runs entirely on-device via Apple Vision Framework. Images never leave your device.

Q: iCloud 同步如何工作？
How does iCloud sync work?

登录同一 Apple ID 的设备之间，笔记通过 CloudKit 端到端加密自动同步。不需要任何第三方账户。右上角的云朵图标显示当前同步状态。

Notes sync automatically across devices signed into the same Apple ID via end-to-end encrypted CloudKit. No third-party account is required. The cloud icon in the top-right shows the current sync status.

Q: 如何搜索笔记？
How do I search notes?

切换到「搜索」标签（Mac 上按 ⌘F），在搜索框输入关键词。应用会搜索 OCR 原文、你的批注、标签、文件夹。

Switch to the Search tab (⌘F on Mac) and type a keyword. The app searches across OCR text, your annotations, tags, and folders.

Q: 如何导出全部数据？
How do I export all my data?

前往「设置」→「数据」→「导出全部数据」（Mac 上按 ⌘E）。应用会生成一个 zip 压缩包，包含 Markdown 格式、JSON 格式和原始图片。

Go to Settings → Data → Export All Data (⌘E on Mac). The app generates a zip archive containing Markdown, JSON, and original images.

Q: Mac 版为什么不能生成 AI 摘要？
Why doesn't the Mac version offer AI summaries?

Mac 版是纯本地优先的截图整理工具，不包含任何 AI 集成。iPhone / iPad 版可选在设置页自备第三方 AI 平台的 API Key。

The Mac version is strictly a local-first screenshot organizer and does not include any AI integration. The iPhone / iPad version optionally supports Bring-Your-Own-Key for third-party AI platforms via Settings.

Q: 关闭主窗口后应用退出了？
The app quits after closing the main window?

是的。截图笔记是单窗口应用，关闭窗口即退出应用——这是 macOS 单窗口应用的标准行为。

Yes. ScreenNote is a single-window app — closing the window quits the app, which is the standard behavior for single-window macOS apps.

Mac 键盘快捷键 / Mac Keyboard Shortcuts

  • ⌘N — 从相册导入 / Import from Photos
  • ⌘F — 搜索 / Search
  • ⌘E — 导出当前笔记 / Export current note
  • ⌘⇧C — 复制摘要 / Copy summary
  • ⌘⌥C — 复制 OCR 原文 / Copy OCR text
  • ⌘⌃C — 复制 Markdown 引用 / Copy Markdown quote
